# COVID-19 Helpbot

Our helpbot connects volunteers and people in need! Our bot is based in the Telegram application

## Built With

* [Telebot](https://pypi.org/project/telebot/) - The bot used
* [Panda](https://pandas.pydata.org/) - To store our .csv files
